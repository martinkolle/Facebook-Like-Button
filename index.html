Facebook Like Button

Facebook Like button for Joomla 1.5, 1.6, 1.7, 2.5 and 3.0

View the Project on GitHub martinkolle/Facebook-Like-Button

Download module

  • Joomla 1.5
  • Joomla 2.5
  • Joomla 3.0

How to use

Firstly install the module to Joomla.

If you are using Open Graph in Joomla 1.6 +, you need to enable the "Facebook like button" plugin.

Open Graph is not supported in joomla 1.5
