Facebook Like button for Joomla

Like button made simple

Why should it be difficult to add a like button to Joomla? I have changed it for you. Simply install and you are ready to get likes!

Joomla 2.5 Joomla 3.0 Old versions

How to install

  1. Download the module.
  2. Install the module in backend
  3. Go to "Extensions" -> "Module" -> "facebook Like".
  4. Choose how you want to use the module under "Settings"
  5. Enjoy a module without credits back to the developer

Know problems

    Hidden comment box

    Solution

    Enable "Z-index" in the module settings or find a new template. (or find a developer to help you)
